# Whitelister
## How to Use?
### Clone the Project
```git clone https://github.com/QuillDev/whitelister.git```
### Install dependencies using yarn or npm
```yarn install```
### Populate the username file
Populate the usernames file with a list of usernames seperated by line
```
QuillDev
Quillewd
DurendalBlade
```
### Run the script
```yarn start```
